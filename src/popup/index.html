MirrorTab mirrors DOM interactions: clicks, inputs, keystrokes, and scrolls from a Source tab to a Target tab in real time. Perfect for demos, testing, and live presentations.

  1. Enter a Source URL and a Target URL.
  2. Click Start Session - two tabs open automatically.
  3. Interact with the SOURCE tab, actions mirror live to the TARGET tab.
  4. Use Pause / Resume or Stop to control the session.
  5. Download the event log as JSON or Text any time.
